Windows安全编程-剪贴板增强器

|Word count:4.3k|Reading time:17min|Post View:

IDE：VS2013

程序运行环境：Win10

编辑于 2021年3月11,12

源代码

程序功能

程序运行后会提供一个窗口，中间是一个剪贴板清单，当对文本进行复制时，复制的内容会依次存入这个剪贴板清单中，并且可以自定义快捷键黏贴指定的内容。

程序窗口:

复制多次使黏贴内容加入剪贴板清单

CreateDialog()函数

GetWindowRect

函数调用成功，则返回值非零，失败则返回值为零

函数功能为获取指定窗口的屏幕坐标

SetWindowPos函数详解

SetWindowLong和SetWindowPos函数详解

关于uFlags的参数：

SWP_NOSIZE = 1 –> 忽略cx, cy， 保持大小

SWP_SHOWWINDOW = $40 –> 显示窗口

函数功能： 改变一个子窗口，弹出式窗口顶层窗口的尺寸，位置和Z序。子窗口，弹出式窗口，及顶层窗口根据它们在屏幕上出现的顺序排序、顶层窗口设置的级别最高，并且被设置为Z序的第一个窗口。

Windows消息机制

主要关注”主程序“和”窗口过程“这两部分，主程序和窗口过程没有直接的调用关系，而在主程序和和窗口过程之间有一个”系统程序模块“。主程序的功能是用来注册窗口类，获取消息和分发消息。而窗口过程的消息则会交给默认的系统过程进行处理。

Windows API——OPENCLIPBOARD——剪贴板

OpenClipboard

打开剪贴板，指定关联到打开的剪贴板的窗口句柄，传入NULL表示当前任务。每次只允许一个进程并访问。没打开一次就要关闭，否则其他进程无法访问剪贴板。

IsClipboardFormatAvailable

判断剪贴板中的数据是什么类型

CF_TEXT：以NULL结尾的ANSI字符集字符串。它在每行末尾包含一个carriage return和linefeed字符，这是最简单的剪贴簿数据格式。

GetClipboardData

执行成功则返回数据句柄，否则返回NULL数据格式，指定格式的数据的句柄

ListView_SetExtendedListViewStyleEx

设置列表样式

Extended List_view Styles

__LVS_EX_FULLROWSELECT__： 选择一个项目后，该项目及其所有子项目将突出显示。此样式仅可与LVS_REPORT样式结合使用。也就是点击一行后，整行都会被选中

__LVS_EX_GRIDLINES__：在项目和子项目周围显示网格线。此样式仅可与LVS_REPORT样式结合使用。

__LVS_EX_HEADERDRAGDROP__：启用列表视图控件中列的拖放重新排序。此样式仅可用于使用LVS_REPORT样式的列表视图控件。

LVS_EX_TWOCLICKACTIVATE：当用户双击某个项目时，列表视图控件会将LVN_ITEMACTIVATE通知代码发送到父窗口。这种样式还可以在列表视图控件中进行热跟踪。热跟踪意味着当光标移至某个项目上方时，该项目将突出显示但未被选中。有关项目激活的讨论，请参见“扩展的列表视图样式备注”部分。

SetClipbordViewer

这个函数会向剪贴板观察链中加入一个观察窗口。当剪贴板的内容发生变化时，该窗口会接收到一个WM_DRAWCLIPBOARD消息。该函数需要传递的参数时观察窗口的句柄。返回值也是一个窗口句柄类型，标识了将要加入的下一个窗口。

WinMain 函数功能

首先调用CeateDialog创建一个MFC窗口并且设置窗口过程函数，之后再调用GetWindowRect获取此窗口再屏幕上的坐标，之后调用SetWindowPos设置此窗口到一个合适的位置，最后监控消息然后发送到窗口过程函数中。

ClipDlgPorc 窗口过程函数的功能

SendMessage/PostMessage 详解

此窗口过程监控消息WM_DRAWCLIPBOARD、WM_HOTKEY、WM_INITDIALOG、WM_COMMAND、WM_CLOSE

1.在WM_DRAWCLIPBOARD消息下会调用OnClipBoardChange函数，这个函数负责打开剪贴板，并判断剪贴板内的数据是否是CF_TEXT型，如果是，则会取出剪贴板内的数据，然后设置剪贴板数据的结构体，在调用InsertIntoList与UpdateListView向list控件中插入数据并更新显示列表信息。

2.在WM_HOTKEY消息下，首先判断是否按下了 Ctrl + Alt 热键，然后根据按下的数字键从列表元素开头取出剪贴板数据（SetClipboardData)，再调用GetCursorPos函数获取鼠标位置，根据鼠标位置获取鼠标所在窗口的句柄，最后通过PostMessage函数向该窗口发送WM_PASTE消息，使剪贴板上的数据黏贴到该窗口中。

3.在WM_INITDIALOG消息下会注册 Ctrl + Alt + 0-9 热键，最后调用SetClipboardViewer函数向剪贴板观察链中加入一个观察窗口，当剪贴板的内容发生变化时，该窗口会接收到一个WM_DRAWCLIPBOARD消息。

4.在WM_COMMAND消息下，监视按键IDC_BTN_CLOSE和IDC_BTN_CLOSE消息，功能分别为关闭程序以及执行测试功能（已移除测试按钮

5.WM_CLOSE，发送终止信号，程序停止运行
